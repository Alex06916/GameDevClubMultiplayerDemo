# GameDevClubMultiplayerDemo
This GitHub respository holds the Unity project for a live demo for the game development club at Clemson University. The project was created using the Unity version 6000.4.0f1.

Instructions for running project locally:
1) Create a Unity account and install Unity Hub at https://unity.com/download
2) Install the Unity Editor version 6000.4.0f1 (latest release at the time of project creation. It can also be downloaded at https://unity.com/releases/editor/archive)
3) Download the .zip of this GitHub repository by pressing the green Code button > Download Zip
4) Extract folder contents
5) Add the project to Unity in the Unity Hub by pressing Add > Add project from disk > Select the GameDevClubMultiplayerDemo-main/MultiplayerDemo folder
6) Open the project (the first launch will take a while since it needs to create the Library folder)
7) The complete project scene/code files can be found in the Assets/Complete game assets folder. The complete game scene can be loaded by double clicking on the Assets/Complete game assets folder/Scenes/CompleteGame scene in the Unity Editor
